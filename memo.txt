std:: :소속을 알리는 네임스페이스 

네임스페이스 : 내부식별자에 범위를 지정하는 선언적 영역으로 여러 라이브러리를 포함할 때 발생할 수 있는 이름 충돌을 방지하기 위해 사용 

std는 standard의 줄임말로 함수, 클래스, 객체 등이 표준 라이브러리에 소속되어 있음을 나타냅니다.

c언어와 달리 cout는 형식지정자 사용 안하고 정수, 문자형, 부동소수 출력 가능 
<< 를 여러개 사용해서 연속해서 출력도 가능 
 ex) cout << "Hello" << 123 << 3.14; 

c언어와 달리 cin은 형식지정자 사용 안하고 입력 가능, 그리고 데이터 형식에 scanf보다 안전  

void 사용 상황
1. 함수가 값을 반환하지 않음을 표시할 때 
2. 함수의 매개변수가 없을 때	
3. 어떤 변수라도 가리킬 수 있는 제너릭 포인터를 만들 때 (형태가 자유로움을 의미)
